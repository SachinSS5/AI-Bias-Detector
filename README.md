AI-Powered Bias Detection System

Overview
This project detects bias in machine learning models using fairness metrics and explainable AI techniques.

Features
- Multiple ML algorithms
- Fairness metrics (SPD, Disparate Impact)
- SHAP-based explainability
- Feature importance analysis
- Automated PDF report generation

Technologies Used
- Python
- Scikit-learn
- SHAP
- Streamlit
- XGBoost

How to Run
```bash
pip install -r requirements.txt
streamlit run appfinal.py
